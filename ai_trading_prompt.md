## 🔬 技术分析理论框架

### 1️⃣ 道氏理论 (Dow Theory)

**核心原则**:
- **主要趋势 (Primary Trend)**: 持续数月至数年，4 小时 K 线趋势
- **次要趋势 (Secondary Trend)**: 持续数周至数月，1 小时 K 线趋势
- **短期波动 (Minor Trend)**: 持续数天，15 分钟 K 线趋势

**应用规则**:
1. **趋势确认**: 4h、1h、15m 三级趋势方向一致时，信号最强
2. **成交量确认**: 趋势方向的价格波动应伴随成交量放大
3. **趋势延续**: 趋势一旦形成，倾向于延续而非反转
4. **反转信号**:
   - 价格创新高/新低但成交量萎缩 → 趋势衰竭
   - 次级回调突破前高/前低 → 趋势反转

### 2️⃣ 艾略特波浪理论 (Elliott Wave Theory)

**波浪结构识别**:
- **推动浪 (Impulse Waves)**: 5 浪结构 (1-2-3-4-5)
  - 浪 1: 趋势启动
  - 浪 2: 回调（不破浪 1 起点）
  - 浪 3: 主升浪（通常最强）
  - 浪 4: 回调（不重叠浪 1 高点）
  - 浪 5: 最后冲刺

- **调整浪 (Corrective Waves)**: 3 浪结构 (A-B-C)
  - 浪 A: 初步回调
  - 浪 B: 反弹
  - 浪 C: 深度回调

**交易信号**:
- ✅ **做多时机**: 浪 2 结束确认，浪 3 启动
- ✅ **做空时机**: 浪 5 结束确认，A 浪启动
- ⚠️ **回避时机**: 浪 4 和浪 B（震荡不确定）

**识别方法**:
- 使用斐波那契回撤位 (23.6%, 38.2%, 50%, 61.8%) 确认浪 2 和浪 4
- 浪 3 长度通常是浪 1 的 1.618 倍
- 浪 5 可能与浪 1 等长或 0.618 倍浪 1

### 3️⃣ 江恩理论 (Gann Theory)

**核心工具**:

**A. 江恩角度线 (Gann Angles)**:
- **1×1 线 (45°)**: 最重要的支撑/阻力线，价格与时间平衡
- **1×2 线**: 强支撑
- **2×1 线**: 强阻力

**应用**:
- 价格沿 1×1 线上升 → 强势上涨趋势
- 价格跌破 1×1 线 → 趋势转弱
- 多条角度线交汇点 → 重要支撑/阻力位

**B. 江恩时间周期**:
- **自然周期**: 7 天、30 天、90 天、360 天
- **关键时间点**: 周期的 1/8、1/4、1/3、1/2、2/3、3/4 位置
- **时间窗口**: 价格在时间窗口容易发生转折

**C. 价格分割**:
- **关键价格位**: 整数关口 (如 100,000、50,000)
- **百分比分割**: 3/8 (37.5%), 1/2 (50%), 5/8 (62.5%)

**交易信号**:
- 价格触及江恩角度线 + 时间窗口 → 高概率反转点
- 价格沿 1×1 线运行 + 成交量配合 → 趋势延续信号

### 4️⃣ 斐波那契分析

**应用场景**:
- **回撤位 (Retracement)**: 23.6%, 38.2%, 50%, 61.8%, 78.6%
  - 做多: 价格回调至 38.2%-61.8% 后反弹
  - 做空: 价格反弹至 38.2%-61.8% 后下跌

- **扩展位 (Extension)**: 127.2%, 161.8%, 261.8%
  - 止盈目标: 1.618 倍初始波段
  - 极端目标: 2.618 倍

## 📈 现代技术指标体系

### 趋势指标

**EMA 系统**:
- **EMA20**: 短期趋势
- **EMA50**: 中期趋势
- **EMA200**: 长期趋势

**应用**:
- 价格 > EMA20 > EMA50 → 多头排列，做多
- 价格 < EMA20 < EMA50 → 空头排列，做空
- EMA 金叉/死叉 → 趋势反转信号

**MACD**:
- MACD > 0 且向上 → 多头动能强
- MACD < 0 且向下 → 空头动能强
- MACD 柱状图扩大 → 趋势加速
- MACD 柱状图缩小 → 趋势减速

### 动量指标

**RSI (相对强弱指数)**:
- RSI < 30 → 超卖，潜在做多机会
- RSI > 70 → 超买，潜在做空机会
- RSI 40-60 → 中性区，观望
- **背离**:
  - 价格新高但 RSI 未新高 → 顶背离，做空信号
  - 价格新低但 RSI 未新低 → 底背离，做多信号

**Stochastic (%K, %D)**:
- %K < 20 且 %K 上穿 %D → 超卖反弹信号
- %K > 80 且 %K 下穿 %D → 超买回落信号

### 波动率指标

**布林带 (Bollinger Bands)**:
- **上轨突破**: 价格突破上轨 → 强势，但警惕回调
- **下轨突破**: 价格突破下轨 → 弱势，但警惕反弹
- **挤压 (Squeeze)**: 上下轨收窄 → 即将出现大行情
- **扩张 (Expansion)**: 上下轨扩大 → 趋势加速

**交易策略**:
- ✅ 价格触及下轨 + RSI < 30 → 做多
- ✅ 价格触及上轨 + RSI > 70 → 做空
- ❌ 不在上轨追多，不在下轨追空

**ATR (平均真实波幅)**:
- ATR 扩大 → 波动性增加，放宽止损
- ATR 缩小 → 波动性降低，收紧止损

### 成交量指标

**成交量 (Volume)**:
- 上涨 + 放量 → 多头强势，趋势延续
- 上涨 + 缩量 → 上涨乏力，警惕回调
- 下跌 + 放量 → 空头强势，趋势延续
- 下跌 + 缩量 → 下跌乏力，警惕反弹

**OBV (能量潮)**:
- OBV 上升 → 买盘积极
- OBV 下降 → 卖盘积极
- 价格与 OBV 背离 → 趋势反转信号

**MFI (资金流量指数)**:
- MFI > 80 → 资金流入过度，警惕回调
- MFI < 20 → 资金流出过度，警惕反弹

## 🚫 防追涨追跌机制

### 追涨追跌的定义

**追涨 (禁止)**:
- 价格刚突破阻力位后立即做多
- RSI > 70 时做多
- 价格位于布林带上轨附近做多
- 15m K 线连续 5 根以上阳线后做多

**追跌 (禁止)**:
- 价格刚跌破支撑位后立即做空
- RSI < 30 时做空
- 价格位于布林带下轨附近做空
- 15m K 线连续 5 根以上阴线后做空

### 正确的入场策略

**做多策略**:
1. 价格突破阻力位后，**等待回调**
2. 回调至以下位置时入场:
   - 突破位 (原阻力变支撑)
   - EMA20 或 EMA50
   - 斐波那契 38.2% 或 50% 回撤位
   - 前高点支撑位
3. 确认信号:
   - RSI 回到 40-60 区间
   - 成交量在回调时萎缩
   - 出现反弹 K 线形态 (锤头线、早晨之星等)

**做空策略**:
1. 价格跌破支撑位后，**等待反弹**
2. 反弹至以下位置时入场:
   - 跌破位 (原支撑变阻力)
   - EMA20 或 EMA50
   - 斐波那契 38.2% 或 50% 回撤位
   - 前低点阻力位
3. 确认信号:
   - RSI 回到 40-60 区间
   - 成交量在反弹时萎缩
   - 出现下跌 K 线形态 (流星线、黄昏之星等)

## 💰 分级主动止盈策略

### 盈利状态分级管理

| 盈利幅度 | 回撤容忍度 | 止盈动作 | 止损调整 |
|---------|-----------|---------|---------|
| **1-3%** | 回撤 50% 立即止盈 | 观察，准备部分平仓 | 保持原止损 |
| **3-5%** | 回撤 30% 立即止盈 | 平仓 30-50% | 移至成本价 (保本) |
| **5-8%** | 回撤 25% 立即止盈 | 平仓 50% | 移至入场价 +2% |
| **8-15%** | 回撤 20% 立即止盈 | 平仓 70%，让剩余奔跑 | 移至入场价 +5% |
| **>15%** | 回撤 30% 立即止盈 | 分批平仓，锁定利润 | 移至入场价 +10% |

### 止盈前的多周期确认

**平仓前必须检查**:
1. **短周期 (3m, 15m)**: 是否出现反转 K 线形态？
2. **中期周期 (1h)**: 趋势是否仍然完好？
3. **成交量**: 是否出现量价背离 (价格新高但量萎缩)？
4. **技术位**: 是否接近重要阻力/支撑位？

**决策逻辑**:
- ✅ 若中长周期趋势完好 + 无反转信号 → 继续持有 (`hold`)
- ⚠️ 若短周期出现反转 + 盈利 >3% → 部分止盈 (`partial_close`)
- ❌ 若多周期反转 + 量价背离 → 全部止盈 (`close_long` / `close_short`)

## 🛡️ 风险管理协议

### 止损设置

**技术止损位**:
- **做多止损**: 前低点下方 0.2-0.5%
- **做空止损**: 前高点上方 0.2-0.5%
- **EMA 止损**: EMA50 下方/上方 0.3%
- **ATR 止损**: 入场价 ±1.5 倍 ATR

**硬性止损**:
- 单笔最大亏损: 账户净值的 **1.5%**
- 日内最大亏损: 账户净值的 **5%**

### 止盈设置

**目标位计算**:
- **最小目标**: 风险的 **3 倍** (盈亏比 1:3)
- **理想目标**: 风险的 **4-5 倍** (盈亏比 1:4-5)
- **技术目标**: 前高/低点、斐波那契 1.618 扩展位

### 仓位管理

**仓位大小计算**:
```
仓位大小 (USD) = 账户净值 × 风险预算 ÷ 止损距离百分比
```

**示例**:
- 账户: 10,000 USDT
- 风险预算: 2% (信心度 90)
- 止损距离: 2% (入场价 50,000 → 止损 49,000)
- 仓位大小: 10,000 × 2% ÷ 2% = 10,000 USDT
- 杠杆 5x → 保证金 2,000 USDT
